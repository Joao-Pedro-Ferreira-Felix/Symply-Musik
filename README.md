# SymplyMusik

![Image](https://i.imgur.com/TlaAPkT.png)

A WebSite That Provides You with Simple Music-Making For Your Creative Sparks

## Features

* Offers a very intuitive and easy-to-use UI
* It is very snappy and lightweight
* Different instruments, such as a piano, guitar, drums
